# 关于我

我是 AI 全栈工程师 / AI 工程实践者，关注 RAG 质量、Agent 可观测性、AI DevOps 与 Vibe Coding。

## 我在做什么

我正在把 AI 应用从“能生成结果”推进到“能被验证和运维”的阶段。这个方向包括：

- RAG 系统的证据、评测和知识治理。
- Agent 执行过程的 trace 回放、失败分析和上下文预算。
- AI DevOps 中的监控、告警、Runbook 和 Incident 闭环。
- 多 AI 接力开发里的项目记忆、交接文档和验证纪律。

## 代表项目

- ProjVault：项目知识中心与 RAG 质量实践。
- NoemaTrace：浏览器内运行的 Agent trace 回放工具。
- OpsMonitor：服务器与中间件自动化监控平台。
- UI-Library：本地优先的 UI 组件资产库与查询系统。
